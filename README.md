### Types of Users
+ Lecturer
  + Login
  + Profile
  + Add a Question
  + Create a Test
  + View Tests Results
  + Logout
+ Student
  + Login
  + Profile
  + View Exams
  + Attempt Exam
  + View Exam Result
  + Logout
